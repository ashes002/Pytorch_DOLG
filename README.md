# Pytorch_DOLG
PyTorch implementation of "DOLG: Single-Stage Image Retrieval with Deep Orthogonal Fusion of Local and Global Features"

reference: https://arxiv.org/pdf/2108.02927.pdf

# Prerequisites
- PyTorch
- PyTorch Lightning
- timm
- sklearn
- pandas
- jpeg4py
- albumentations
- python3
- CUDA
